# 특이사항

- 버튼을 누르면 아이템 창이 새로 생성되고 닫기 누르면 없어짐.
- 헤더부분 화면을 sflight, 아이템부분 화면을 sbook 테이블을 참고하였음
- 화면 0100의 status BACK,EXIT,CANC를 E커멘드로 하였음
